## Ответы на контрольные вопросы к лабораторной работе №2:

**1. Какие есть типы целей и чем они отличаются?**

**Существует два типа целей:**

* **Цели пользователя:** Описывают, чего хочет добиться пользователь, используя продукт.
* **Бизнес-цели:** Описывают, чего хочет добиться компания, создавая продукт.

**Цели пользователя и бизнес-цели должны быть взаимосвязаны.** Успешный продукт должен решать задачи пользователей и одновременно приносить пользу компании.

**2. Чем отличается цель от задач?**

**Цель:** это конечный результат, к которому стремится пользователь или компания.

**Задача:** это конкретное действие, которое необходимо выполнить, чтобы достичь цели.

**Цель более абстрактна, чем задача.** Задачи - это шаги, которые необходимо предпринять для достижения цели.

**3. Для чего проводится обзор аналогов?**

Обзор аналогов проводится, чтобы:

* **Изучить существующие решения на рынке.**
* **Понять, как конкуренты решают задачи пользователей.**
* **Найти идеи для своего продукта.**
* **Избежать ошибок, которые уже сделали другие.**

**4. Из каких шагов состоит общий план обзора аналогов?**

**Общий план обзора аналогов:**

1. **Определить список конкурентов.**
2. **Составить список критериев для сравнения.**
3. **Проанализировать каждый конкурент по списку критериев.**
4. **Сделать выводы и составить список рекомендаций.**

**5. Что такое референсы и для чего они нужны?**

**Референсы:** это примеры работ, которые могут быть использованы для вдохновения или для заимствования идей.

**Референсы нужны:**

* **Чтобы собрать идеи для своего проекта.**
* **Чтобы понять, как другие дизайнеры решают задачи.**
* **Чтобы создать мудборд.**

**6. Что такое мудборд?**

**Мудборд:** это визуальная презентация, которая представляет собой коллаж из изображений, текста, цветов, шрифтов и других элементов.

**Мудборд используется:**

* **Чтобы визуализировать идеи проекта.**
* **Чтобы донести свою идею до других людей.**
* **Чтобы создать единый стиль для проекта.**

**7. Какая цель использования мудборда?**

**Цель использования мудборда:**

* **Собрать идеи и вдохновение.**
* **Определить стиль проекта.**
* **Создать единый язык для команды.**
* **Продемонстрировать проект клиенту.**

**Мудборд - это эффективный инструмент, который помогает дизайнерам создавать красивые и функциональные продукты.**
